# Contains Profanity

**Tests:** Content moderation, profanity detection, language appropriateness assessment

**Response type:** Boolean (Yes/No)

---

Does this audio message contain profanity?

Respond with only: **Yes** or **No**

Answer **Yes** if:
- The speaker uses profane, vulgar, or obscene language
- Swear words, curse words, or strong expletives are present
- The recording contains language that would typically be censored in broadcast media
- Slurs or highly offensive terms are used

Answer **No** if:
- The language is clean and suitable for all audiences
- The speaker uses mild expressions that are not considered profanity (darn, heck, shoot, etc.)
- No vulgar, obscene, or offensive language is present
- The recording contains only professional or casual but clean language
